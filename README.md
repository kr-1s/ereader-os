# ereader-os

add all books in python-ereader/books
  currently, pdfs read right to left instead of left to right, since im mainly only using pdfs to read manga
  when i add a settings page ill probably add toggling between right to left and left to right
if modifying screensaver or the statistics image, the name of the image must remain the same
  ex. screensaver.jpg must remain named screensaver.jpg even if the image itself changes
the trackers track statistics and bookmarks. bookmarks are automatically tracked when you leave a book, and both json files are created if there are none present on opening of the program
its built for an eink display, which is why there are black flash transitions
